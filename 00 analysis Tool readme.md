### 分析工具

1. 选择文件夹

2. 确认该文件夹名称是否为相机sn类型：

   + 如果是，则处理本文件夹下的内容，分析总结为一个结结果
   + 如果不是，则扫描文件夹下的子文件夹列表，对于每个子文件夹进行判断，是否满足sn命名规则，满足则当作一组数据处理；不满足则跳过；最终，总结所有满足规则的项目，作为结果
   + 结果保存在选择文件夹的根目录；命名为results.xlsx

3. 在扫描完成所有需要处理的文件后，跳出第一组的power=100的图像，用于框选ROI计算范围；

   + 可能存在left\right 各一组，那么分别进行一次ROI选择的过程，然后出两种结果

   + 结果中，left对应left，right数据对应right

4. 根据框选的ROI范围，计算每组的结果；如果存在left/ right，数据分别对应；

5. 关于结果： 

   + 结果的分析方法：
     + 目标文件夹内包含n张原始图；
     + 原始图命名规则为： left/right_e[expo]_p[idx].png；其中left 、 right 为组类型；idx为laserPowerValue；idx为三位数的是真正开启的激光并设置laserPower为idx对应的值，如000 为laser设置为0的状态，还有四位数的0000，代表laseren = false，是真正的无光状态，绘图时，该点映射为laser=-1；
     + 结果计算过程，分别对每张图进行处理，提前ROI范围内的亮度均值，对应laserPowerValue，形成结果图表
     + 另外，统计每张图ROI范围内的过曝点数量占ROI范围的总点数的 比例，出结果可靠性评估表

   + 结果以图表方式绘制结果，横坐标为 laserPowerValue 纵坐标为亮度；
   + 包含sheet： 
     + 结果sheet：包含left/right 结果图表；结果可靠性评估表；一共2/4张
     + 原始left数据
     + 原始right数据
